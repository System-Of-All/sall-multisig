# Contributing to SALL Multi-Sig Wallet

Thank you for your interest in contributing to the SALL Multi-Sig Wallet project.  
This document describes how to contribute, what to expect, and how contributions are governed.

## How to Contribute

1. Fork the repository
2. Create a branch with a clear name
3. Submit a pull request
4. Align with CODEOWNERS review
5. Follow contribution tracks and tests

## Expectations

- Follow the coding and documentation standards
- Write clear commit messages
- Provide test cases when relevant
- Remain aligned with architectural invariants
